4.Use man top
  El comando top muestra un resumen del estado de nuestro sistema y la lista de procesos que se estan ejecutando.
  PID: identificador del proceso
  User: usuario propietario del proceso
  PR: prioridad del prceso
  NI: asigna la prioridad
  Virt: cantidad de memoria virtual utilizada
  Res: cantidad de memoria RAM fisica q utiliza el proceso
  SHR: memoria compartida
  S: estado del proceso
  CPU: porcentaje de cpu utilizada
  MEM: porcentaje de memoria fisica utilizada
